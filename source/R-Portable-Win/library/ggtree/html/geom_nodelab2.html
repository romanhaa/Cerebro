geom_nodelab2 {ggtree}R Documentation

@geom_nodelab2

Description

add node label for circular layout

Usage

geom_nodelab2(mapping = NULL, nudge_x = 0, nudge_y = 0,
  geom = "text", hjust = 0.5, ...)

Arguments

mapping

aes mapping

nudge_x

horizontal adjustment to nudge label

nudge_y

vertical adjustment to nudge label

geom

one of 'text', 'label', 'image' and 'phylopic'

hjust

horizontal alignment, one of 0, 0.5 or 1

...

additional parameters

Value

node label layer

Author(s)

Guangchuang Yu
